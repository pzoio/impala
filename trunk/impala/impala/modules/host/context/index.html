Welcome to the Impala web application

This is the Hello World of Impala, just to help you get started. It consists of the following:

  • A root or main module, which contains interfaces such as MessageService and core domain classes.
  • A single implementation module, which contains an implementation for MessageService and exports this to Impala's service registry.
  • A single web module, which uses the message service to get a message to print out to the screen.
