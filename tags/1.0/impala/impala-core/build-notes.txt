Targets:

Jar build:

-clean
Remove any built jar

-compile-src
Compiles all source folders to common output folder
Params: source directories
Params: resource directories

-jar
Creates a jar of source folders

-compiles-test
Compiles all test to common test output folder
Params: source directories
Params: resource directories

-run-test
Runs test, as specified by a pattern
